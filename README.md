# Nycolas Rocha Portfolio

Personal portfolio built with Next.js, React, TypeScript, and a game-inspired interface.

## Scripts

```bash
npm run dev
npm run lint
npm run build
```

Open [http://localhost:3000](http://localhost:3000) during development.

## Analytics

Google Analytics 4 and Vercel Analytics are configured. See [ANALYTICS_SETUP.md](./ANALYTICS_SETUP.md) for details.

## Main Files

- `src/app/HomeClient.tsx` - interactive portfolio UI
- `src/app/home-content.ts` - Portuguese and English content
- `src/app/globals.css` - visual system and responsive styles
- `src/app/GoogleAnalytics.tsx` - GA4 integration
